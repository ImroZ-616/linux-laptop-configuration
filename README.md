# linux-laptop-configuration

This repository serves as a journal documenting the changes, configurations, and software installations I make while transitioning from Windows 11 to a dual-boot setup with Ubuntu 22.04 LTS.

The purpose of this repository is to keep track of my setup process, notes, commands, and experiences so that I can easily reproduce the configuration in the future or assist others following a similar path .

## Contents

* `obsidian.md` — Notes and steps taken to install and configure Obsidian on Ubuntu.
